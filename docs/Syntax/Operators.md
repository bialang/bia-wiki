# Operators

## Operator Precedence

| Precedence | Operator            | Description                            |
| :--------: | ------------------- | -------------------------------------- |
|     1      | ()                  | Parentheses                            |
|     2      | foo()<br/>          | Function call                          |
|            | .                   | Member access                          |
|            | `**`                | Exponentiation                         |
|            | `*`, `/`, `//`, `%` | Multiplication, division and remainder |
|            | `+`, `-`            | Addition and subtraction               |
|            | `not` *value*       | Boolean negation                       |
|            | `and`               | Boolean conjunction                    |
|            | `or`                | Boolean disjunction                    |
